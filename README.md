# MySTDF_Tools

MySTDF_Tools is a two-stage STDF processing and analysis toolchain
for Windows, consisting of:

- **My_STDF** – STDF → SQLite database converter
- **MySTDF_Pro** – SQLite database → analysis & visualization tool

This repository contains **source code only**.
Precompiled Windows executables are provided via **GitHub Releases**.

------------------------------------------------------------
Overview
------------------------------------------------------------
The tools are designed to be used together in the following workflow:

    STDF (*.stdf / *.std)
          ↓
        My_STDF
          ↓
    SQLite DB (*.db)
          ↓
      MySTDF_Pro
          ↓
  Analysis / Wafer Maps / Plots / Reports

------------------------------------------------------------
Tools
------------------------------------------------------------

### 1. My_STDF (Front-End Converter)
- Converts raw STDF files into a SQLite database
- May also generate Excel files and logs
- Intended as the first step in the workflow

### 2. MySTDF_Pro (Analysis Tool)
- Loads SQLite DB files generated by My_STDF
- Performs:
  - Wafer / bin maps
  - Parametric distributions
  - Correlation and sanity checks
- Does NOT load raw STDF files directly

------------------------------------------------------------
Download (Windows Standalone)
------------------------------------------------------------
Precompiled Windows (x64) binaries are available from the
**GitHub Releases** page.

Each release includes:
- My_STDF (STDF → DB)
- MySTDF_Pro (DB → analysis)
